# CoffeeMatch / SoulMate
![SoulWork screenshot](https://raw.githubusercontent.com/kkoeckritz/koecimg/master/soulmate-screen.png)
[Visit the live demo â†—](http://soulwork-soulmate-demo.herokuapp.com)

**CoffeeMatch** contains the open source engine behind the Soul Work SoulMate roast matching quiz.\
**SoulMate** has been carefully designed for Soul Work Coffee, and in its current state may not be suited for more generic applications. Its source is provided here as a reference for MERN developers seeking a live example.
